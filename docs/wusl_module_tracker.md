## **GPA CALCULATION GUIDE**

### Grade Point Values
| Grade | Range | Points |
|-------|-------|--------|
| A+ | 85-100 | 4.0 |
| A | 70-84 | 4.0 |
| A- | 65-69 | 3.7 |
| B+ | 60-64 | 3.3 |
| B | 55-59 | 3.0 |
| B- | 50-54 | 2.7 |
| C+ | 45-49 | 2.3 |
| C | 40-44 | 2.0 |
| C- | 35-39 | 1.7 |
| D+ | 30-34 | 1.3 |
| D | 25-29 | 1.0 |
| E | 0-24 | 0.0 |

### Formula
**GPA = Σ(Credits × Grade Points) ÷ Σ(Credits)**

### Example Calculation
If you completed:
- Module 1: 3 credits, Grade A (4.0) = 12.0
- Module 2: 4 credits, Grade B+ (3.3) = 13.2
- Module 3: 2 credits, Grade B (3.0) = 6.0

**Total:** (12.0 + 13.2 + 6.0) ÷ (3 + 4 + 2) = 31.2 ÷ 9 = **3.47 GPA**
